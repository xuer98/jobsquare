Position Summary
Duties: Design, engineer, develop, and maintain the core platform software applications at company to enable the engineering teams to safely and reliably test and release code into production software systems. Design, engineer and implement the backend API endpoints and the frontend user interface for the web applications that support the core platform software applications using technologies including, Java, Angular, and REST services. Debug software issues and provide software engineering solutions for other engineering teams. Monitor and analyze the software system metrics to ensure the reliability of the services and troubleshoot performance bottleneck. Investigate and debug the quality assurance results of the production software code to resolve the software exceptions. Conduct software code reviews and design code reviews with fellow software engineers. Collaborate with other software engineers to create the engineering design and roadmap for the team’s software products used internally by the company.

Minimum education required: Bachelor’s Degree in Computer Science or related Information Technology field.   

Skills required: Must have knowledge of the following software skills and technologies: programming in Python, Java, and BASH/Shell Scripting; developing/engineering backend services using REST API endpoints, gRPC frameworks, Swagger, and Protobuf; database management systems including PostgreSQL, Microsoft SQL, and Amazon S3; ability to engineer/develop frontend web interfaces using React (or Angular), JavaScript, CSS, and HTML; data streaming and orchestration technologies including Kafka and Airflow; cloud services hosted on AWS and Kubernetes; Linux systems and Linux commands; observability tools (Grafana or Datadog); modern IDE (VScode, IntelliJ), version control tools (GIT), and deployment pipeline (Jenkins); automating secure remote administration using Paramiko for SSH connections, including key management, parallel command execution, SFTP file transfer; using Supervisord and advanced Linux process management tools to ensure stable and continuous trading operations, including health checks and log rotation; implementing and maintaining observability for trading backend systems using the ELK stack (Elasticsearch, Logstash, Kibana) and StatsD; designing, deploying, and maintaining containerized architectures for equity trading backend systems using Kubernetes Client Library for workload orchestration and Terraform, Helm for automated infrastructure provisioning across high performance compute clusters; orchestrating and automating backend workflows using Airflow, Cron, Temporal, and Bonita (BonitaSoft); developing and operating data streaming pipelines supporting trading backend operations utilizing Kafka, Celery, Redis, and applying Protobuf serialization for efficient, reliable exchange of real time trading system data; orchestrating and executing Apache Spark jobs for large scale distributed data processing for trading datasets; designing and implementing gRPC based microservices for low latency inter service communication; configuring secure messaging and connectivity for trading systems using Solace and VPN technologies; implementing MS Graph API integrations for email automation and cloud workflows; designing and developing internal web dashboards using React and the Ant Design as the primary UI framework; fine tuning and optimizing MSSQL databases supporting transactional trading data, including execution plan analysis, query optimization, indexing and partitioning strategies, and performance tuning under high load conditions while ensuring ACID consistency; engineering backend services and data access layers using SQLAlchemy, including ORM modeling, migrations, session/connection pooling; implementing, running, and orchestrating large scale Deterministic and Stochastic Simulations for models. Must also pass the company's required skills assessment. 

Base salary: The base pay for this role will be between $165,000 and $325,000 per year. This role may also be eligible for other forms of compensation and benefits, such as a discretionary bonus, health, dental and other wellness plans and 401(k) contributions. Discretionary bonus can be a significant portion of total compensation. Actual compensation for successful candidates will be carefully determined based on a number of factors, including their skills, qualifications and experience.
